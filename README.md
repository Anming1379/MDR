## 介绍

用来浏览 markdown 文件的, 我一般用来快速发布 markdown (

(窝补会代码是用AI写的)

---

## 用法

#### 想快速打开本地文件?

- 下载 Markdown Reader.html

- 直接打开这个网页, 可以用来浏览本地网页

#### 想直接加载设置好的文件?

- 下载这些:
  - index.html
  - build-manifest.js
  - 提取Text.bat

- 创建一个名为 Text 的文件夹, 并把你的 markdown 文件放进去. (支持文件夹和 txt )

- 运行 提取Text.bat ,会生成一个叫 manifest.js 的文件

- 打开 index.html 就可以啦
